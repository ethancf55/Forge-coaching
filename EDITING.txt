WORKFLOW — edit locally, publish to Vercel
------------------------------------------
  1. Open this folder in VS Code:
       File → Open Folder → forge-coaching

  2. Edit files and save.

  3. Preview locally (optional):
       Right-click index.html → Open with Live Server
       OR in terminal: python3 -m http.server 8080
       Then visit http://localhost:8080

  4. Deploy to Vercel (pick one):
       • VS Code: Vercel extension → Deploy
       • Terminal: vercel --prod   (from this folder)

  5. Wait ~30 seconds, then check https://forgevirtualtraining.org

CONTACT FORM EMAIL SETUP (one-time)
-----------------------------------
  The form uses Vercel serverless: api/contact.js
  Emails are sent through Resend (free tier).

  1. Sign up at https://resend.com
  2. Create an API key (Resend dashboard → API Keys)
  3. In Vercel: Project → Settings → Environment Variables
     Add these for Production (and Preview if you want):

     RESEND_API_KEY     = your Resend API key
     NOTIFY_EMAIL       = ethanflem1@gmail.com
     FROM_EMAIL         = Forge Coaching <onboarding@resend.dev>

     (Later, after verifying your domain in Resend, change FROM_EMAIL to:
      Forge Coaching <hello@forgevirtualtraining.org>)

  4. Redeploy the site after adding env vars
  5. Test the form — you should get a lead email AND the client gets a confirmation email
